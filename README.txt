TROUBLESHOOTING:
----------------
If the palette doesn't appear or elements look broken:
1. Make sure you're opening the file in a modern browser
   (Chrome, Firefox, Safari, Edge)
2. Don't rename or move files from the "assets" folder
3. Keep all files together in the same directory structure
